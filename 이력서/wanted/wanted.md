한국조형예술고등학교 그래픽디자인과에서 정밀묘사 등 시각 표현의 기초를 다지고, 울산대학교 디지털콘텐츠디자인을 전공하며 가독성을 위한 행간 설계, 요소의 부피·무게에 따른 레이아웃 배치 등 시각 디자인 원리를 체화했습니다. 실제 유저 대상 사용성 테스트를 촬영·분석하여 UX 개선점을 도출한 경험이 있으며, 졸업 후 바로 프론트엔드 개발자로 취업하여 디자인 시스템 구축, 컴포넌트 라이브러리 설계, 반응형 UI 구현까지 디자이너의 눈과 개발자의 손을 동시에 활용해 제품의 UX 품질을 높여왔습니다.

대학 시절에는 디자인 굿즈 브랜드를 공동 창업하여 월 천만원 매출을 달성하고 해외 판매를 주도한 경험이 있습니다.

- 디자인 시스템 구축: Atomic Design 기반 컴포넌트 라이브러리 설계, Typography·트랜지션 값 중앙 정의로 브랜드 경험 일관성 확보
- 주도적 문제 정의 및 해결: 성능 병목 자체 발견 및 78.75% 로딩 개선, 인프라 전환 제안·주도, 협업 개선을 위한 Figma 플러그인 자체 개발

[인핸드플러스] 프론트엔드 개발자 (2022.06 ~ 2025.11, 3년 6개월)
헬스케어 SaaS — B2C 투약 순응도 모니터링 서비스

1. 페이지 로딩 78.75% 개선 (8초 → 1.7초)
서비스 페이지가 평균 8초 걸리는 것을 아무도 문제 삼지 않는 상황에서, 스스로 문제를 정의하고 2단계에 걸쳐 해결했습니다. 1차로 번들 분석 후 코드 스플리팅·트리 셰이킹으로 3초까지 줄였고(62.5% 개선), 2차로 AWS ECS에서 Vercel Edge Network 서울 리전으로 전환하여 1.7초까지 추가 개선했습니다(43.3%).

2. Next.js 메모리 사용량 40% 감소 — 502 장애 해결
운영 중 간헐적으로 502 Bad Gateway가 발생했습니다. 서버 로그와 메모리 프로파일링을 통해 axios 인스턴스의 메모리 누수가 근본 원인임을 파악하고, HTTP/2 통신 개선과 함께 해결하여 메모리 사용량을 40% 줄이고 장애를 완전히 해소했습니다.

3. ECS → Vercel 전환 제안·주도 — 연 450만원 절감
성능 최적화 과정에서 AWS ECS의 비용 대비 효율이 낮다는 점을 파악하고, Vercel 전환을 팀에 제안했습니다. PoC를 직접 진행하여 성능·비용 비교 데이터를 만들고, 전환을 주도하여 연간 약 450만원의 인프라 비용을 절감했습니다.

4. 모바일 웹·데스크톱·네이티브 웹뷰 3환경 동시 서비스
기존 데스크톱 전용 서비스를 Next.js 기반 반응형으로 전면 리뉴얼했습니다(입사 6개월 만에 주도). 폰/태블릿/PC 3단계 브레이크포인트를 설계하고, 이후 Android(Kotlin)/iOS(Swift) 네이티브 앱의 WebView에 동일 웹을 탑재하여 하나의 코드베이스로 3가지 환경을 동시에 서비스하는 구조를 만들었습니다. Android/iOS 네이티브 앱 빌드·배포(Lottie 애니메이션 스플래시 포함)도 직접 담당했습니다.

5. 디자인 시스템 구축 및 운영
Figma Dev Mode에서 디자인 스펙을 확인하며 Typography 컴포넌트를 설계했습니다. props로 폰트사이즈/자간/행간/시맨틱태그를 제어하고, TypeScript 타입 강제로 잘못된 스타일 조합을 런타임 전에 차단하는 구조입니다. 트랜지션 값(0.3s ease-in-out 등)을 중앙 정의하여 서비스 전체의 브랜드 경험 일관성을 자발적으로 확보했습니다.

6. 아키텍처 선택의 트레이드오프 판단
프로젝트 규모가 커지면서 Feature-Sliced Design을 학습·도입했으나, 작은 UI 변경에도 수정해야 할 레이어가 많아 빠른 실험·배포 환경에서 생산성이 오히려 떨어지는 것을 확인했습니다. 트레이드오프를 분석한 끝에 디자인 시스템과의 정합성이 높은 Atomic Design으로 전환하는 결정을 내렸습니다. 단순히 기술을 도입하는 것이 아니라, 팀과 제품에 맞는지 판단하고 방향을 수정할 수 있는 경험이었습니다.

7. 4개 국어 국제화 시스템 구축
next-i18next 기반으로 SSR/SSG 환경의 국제화를 설계하고, Accept-Language 헤더로 자동 언어 감지를 구현했습니다. 노션 데이터베이스와 번역 파일을 자동 동기화하는 시스템을 개발하여 기획자·QA·개발자 간 번역 협업 워크플로를 구축했습니다.

8. 데이터 시각화 및 백오피스 UX 설계
Chart.js 기반 투약 순응도/걸음수 그래프에서, 60개 이상 데이터 포인트 시 자동 생략 처리 알고리즘을 설계하여 스크롤 없이 핵심 정보를 전달하는 대시보드를 완성했습니다. 백오피스에서는 실시간 자동 검색, 멀티 필터링, 퀵서치를 설계·구현하여 관리자의 정보 탐색 효율을 개선했습니다.

9. Figma Plugin 개발 및 테스트 자동화
디자인 규격서 변경 사항을 개발자가 수동으로 추적해야 하는 문제를 해결하기 위해, Figma Plugin API를 활용한 규격서 자동 추적 도구를 직접 개발하여 팀에 제안했습니다. 또한 Cypress 기반 E2E 테스팅 환경을 구축하고, data-cy 속성으로 테스트 타겟을 표준화하며, 테스트 실패 시 Slack 자동 알림을 연동하는 품질 관리 프로세스를 만들었습니다.
