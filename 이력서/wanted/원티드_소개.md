# 원티드 간단소개 (토스증권 Frontend UX Engineer)

> 아래 텍스트를 원티드 "간단소개"에 붙여넣기 (약 1,800자)

---

디자인 전공 출신 프론트엔드 개발자입니다 (4.5년 경력).

한국조형예술고등학교 그래픽디자인과에서 정밀묘사·채색 등 시각 표현의 기초를 다지고, 울산대학교 디지털콘텐츠디자인을 전공하며 가독성을 위한 행간 설계, 요소의 부피·무게에 따른 레이아웃 배치 등 시각 디자인 원리를 체화했습니다. 졸업 후 바로 프론트엔드 개발자로 취업하여, 디자이너의 눈과 개발자의 손을 동시에 활용해 제품의 UX 품질을 높여왔습니다.


[반응형 디자인 & 웹 접근성]

하나의 코드베이스로 폰/태블릿/PC/네이티브 앱 웹뷰까지 전환경을 대응한 경험이 있습니다. Next.js 기반으로 3단계 브레이크포인트를 설계하고, Android(Kotlin)/iOS(Swift) 네이티브 앱의 WebView에 동일한 웹을 탑재하여 별도 네이티브 UI 개발 없이 앱 사용자 경험을 제공했습니다.

폼 에러 발생 시 해당 필드로 포커스·스크롤이 자동 이동하는 UX를 섬세하게 구성했고, 트랜지션 값(0.3s ease-in-out 등)을 중앙에서 정의·전파하여 서비스 전체에 일관된 브랜드 경험과 예측 가능한 인터랙션을 자발적으로 구축했습니다.


[디자인 시스템 구축/유지보수]

두 회사에서 Atomic Design 기반 컴포넌트 라이브러리를 설계·운영했습니다. Typography 컴포넌트를 props로 폰트사이즈/자간/행간/시맨틱태그를 제어할 수 있도록 설계하고, TypeScript 타입 강제로 잘못된 스타일 조합을 런타임 전에 차단했습니다. 4-point grid system을 적용해 픽셀 깨짐을 방지하고, 디자이너가 정의한 스펙이 코드에 일관되게 반영되는 환경을 만들었습니다.

디자인-개발 간 협업 병목을 해소하기 위해 Figma Plugin API를 활용한 자동화 도구를 직접 개발해 팀에 제안한 경험도 있습니다.


[공통 모듈/라이브러리 설계]

재사용 가능한 Modal·Chart 등 공통 UI 모듈을 개발하고, Sass/SCSS 아키텍처(base/constants/layouts/mixins/modules)를 설계했습니다. next-i18next 기반 4개 국어 국제화 시스템을 구축하며 노션 데이터베이스와 번역 파일 자동 동기화 시스템도 개발했습니다.
