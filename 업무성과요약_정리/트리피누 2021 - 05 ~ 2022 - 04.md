# 트리피누 2021 - 05 ~ 2022 - 04

---

트리피누 (여행 플랫폼 스타트업) - 2021.05~2022.04 (1년)

핵심 성과

- 220개국 대응 여행일정 루트 시스템 개발: react-leaflet 기반 지도 기능 구축,
순서가 있는 커스텀 마커 아이콘 동적 생성
- 패키지 여행 예약 시스템 구축: 회원/비회원 예약 기능, 이벤트 관리, 다중
화폐(KRW/USD) 지원, 관리자 페이지 개발
- Atomic Design Pattern 기반 디자인 시스템 구축: Atoms/Molecules/Organisms 계층
구조, TypeScript로 타입 안전성 확보, Storybook 통합
- 반응형 웹 설계: BEM 방법론, Material-UI Grid, Sass/SCSS
아키텍처(base/constants/mixins/modules), mobile/tablet/desktop 대응
- 국제화(i18n) 구현: react-i18next 기반 브라우저 언어 자동 감지, 한/영 지원,
동적 콘텐츠 번역
- 호텔 예약 시스템: [Booking.com](http://booking.com/) API 연동, 다국어 호텔명 관리, 위치 기반 검색
- 시드 투자 유치 및 수상: 비즈니스 성과 달성 ([http://www.tripinu.com/](http://www.tripinu.com/))

주요 기술 스택

- Frontend: React, TypeScript, JavaScript
- Styling: Sass/SCSS, emotion (CSS-in-JS), Material-UI
- State Management: Recoil
- Maps: react-leaflet (OpenStreetMap)
- i18n: react-i18next
- Data Fetching: axios, SWR
- Form: React Hook Form, Antd
- Tools: Storybook, Prettier, ESLint
- Libraries: react-router-dom, react-responsive, Day.js, date-fns, Swiper,
uuid, Gravatar
- Architecture: Atomic Design Pattern, BEM methodology

상세 개발 내역

1. 여행 지도 기능 (react-leaflet)
- 220개국 여행 경로 시각화 시스템 개발
- 동적 마커 아이콘 생성 로직 구현 (day 1~15 순차 번호 표시)
// 마커 이미지를 순서에 맞게 동적 생성
let markerIconGenerator = () => {
for (let i = 1; i <= markerPositions.length; i++) {
let icon = new Icon({
iconUrl: `./images/marker_${i}.png`,
iconSize: [70, 70],

markerIconStorage.push(icon);
}
};
- Public 폴더 활용으로 15개 이상의 import 없이 이미지 동적 로드
- 최소 15개 핀 표시 지원, 여행지 수에 따라 동적 마커 생성
1. 패키지 여행/예약 시스템
- Event 관리: 이벤트 생성, 만료일 관리, 인원/요금 설정, 상태 관리
- Reservation 시스템:
    - 회원/비회원 구분 예약 (booking_id 자동 생성: k211122173049@001012345678
    형식)
    - 예약 타입, 예약 인원, 대표자 정보 관리
    - 계약금/잔금/환불 처리 및 날짜 추적
    - 예약 확인/취소 기능
- 관리자 페이지: 예약 목록 조회, 사용자 정보 관리, full_name 조합 표시
1. 호텔 시스템
- [Booking.com](http://booking.com/) API 연동으로 호텔 데이터 통합
- Geolocation 문자열 파싱 유틸리티 구현:
const stringToObject = (stringGeo) => {
const start = stringGeo.indexOf('{') + 1;
const end = stringGeo.indexOf(',');
const getLet = stringGeo.slice(start, end);
const getLng = stringGeo.slice(end + 2, -1);
return { lat: parseFloat(getLet), lng: parseFloat(getLng) };
};
- 다국어 호텔명 관리 (hotel_name, hotel_name_ko)
- 다중 화폐 지원 (KRW, USD), 호텔 등급/리뷰 관리
1. Atomic Design Pattern 디자인 시스템
- Atoms: Input, Typo (텍스트 스타일 시스템), Text, FlexBox 등 최소 단위
컴포넌트
- Molecules: Card, Loading, Modal, Backdrop 등 조합 컴포넌트
- Organisms: Footer, GNB (Global Navigation Bar), Sidebar 등 복합 컴포넌트
- Pages: Template 레이아웃 + 컴포넌트 주입으로 완성
- Typo 컴포넌트: Figma 디자인 시스템 기반, props로
폰트사이즈/자간/행간/시맨틱태그/마진/패딩 제어
- TypeScript로 props 타입 강제하여 런타임 전 디버깅
- 8-point grid system 적용 (픽셀 깨짐 방지)
- Storybook으로 컴포넌트 문서화
1. 반응형 웹 설계
- BEM 방법론 적용: block__element--modifier 네이밍
- Material-UI Grid 시스템 활용
- Sass/SCSS 아키텍처:
    - base: 폰트, CSS 리셋 등 전역 스타일
    - constants: 폰트 크기, 색상, z-index, breakpoint 상수
    - layouts: Grid 시스템, overlay
    - mixins: flex, 텍스트, position 중앙정렬, ellipsis
    - modules: button, card, input form 스타일
    - main.scss: 전체 통합 파일
- react-responsive 라이브러리로 디바이스별 분기 처리
1. 국제화(i18n) 구현
- react-i18next 통합 및 브라우저 언어 자동 감지
const { t, i18n } = useTranslation();
const isKorean = t('serch_language') === 'city_name_ko' ? true : false;
- 다국어 JSON 파일 관리 (src/lang/translation.ko.json)
- 동적 필드명 변환 (city_name → city_name_ko)
1. Custom Hooks 개발
- commaThousand: 가격 천단위 쉼표 표기
export const commaThousand = (price) => {
if (Number.isInteger(price)) {
return price?.toString().replace(/\B(?=(\d{3})+(?!\d))/g, ',');
} else {
// 소수점 달러 처리
return price?.toString().replace(/\B(?<!\.\d*)(?=(\d{3})+(?!\d))/g, ',');
}
};
- 타입 체크 및 예외 처리 (undefined, null, string 처리)
- 국가별 천단위 표기 차이 고려 (미국/영국 vs 유럽)
1. 코드 품질 & 아키텍처
- 파일 구조:
    - components: atoms/molecules/organisms (Atomic Pattern)
    - pages: 페이지별 컴포넌트 (index.jsx/tsx가 최종 페이지)
    - hooks: 커스텀 훅 관리
    - lang: i18n JSON 파일
    - Layout: GNB, Footer 조합 및 렌더링
    - styles: Sass 구조 (base/constants/layouts/mixins/modules)
- CamelCase 네이밍 컨벤션 (언더바 금지)
- 상태 관리: Recoil로 전역 상태, Page 레벨에서 로컬 상태 관리 후 props drilling
- TypeScript 마이그레이션: 복잡한 상태 관리 파일을 .tsx로 전환
- 인수인계 문서 작성: 시스템 configuration, 소스 구조, 스타일링 방법 상세
문서화
1. 성능 & UX 최적화
- 이미지 최적화: preload 기능 구현
- 로딩 상태: Backdrop + axios로 POST/PATCH 시 로딩 표시
- 모달 시스템: 재사용 가능한 Modal 컴포넌트
- 미저장 방지: 뒤로가기/새로고침 시 경고 기능
- 로그인 플로우: 로그인 후 원래 페이지 복귀 (state { from })
- SEO 최적화: 검색 엔진 최적화 적용
- Google Analytics: 트래킹 코드 삽입, 클릭율/이탈율 측정
1. 기타 기술적 구현
- 정규표현식: 폼 검증 (전화번호, 생년월일, 이메일 등)
- react-router-dom: useParams, Query String 처리
- Google 거리 계산 API 연동
- Amadeus API: 공항 검색 기능
- Viator API: 가이드 위젯 연동
- ReactToPrint: 웹 화면 인쇄 기능
- Gravatar: 사용자 프로필 이미지
- faker-js: 테스트용 더미 데이터 생성
- Swiper: 캐러셀 UI 구현
- 환율 계산: USD/KRW 환전 기능
