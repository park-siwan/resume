통합관리자 시스템 개발 (7월~12월)

메시징 시스템 구축

- 수동 메시지 시스템 개발 (액션 버튼, 폼 검증)
- 자동 메시지 시스템 개발 (예약발송, 반복발송 기능)
- 발송 내역 페이지 개발:
    - DataGrid 한글화
    - 검색 기능 추가
    - React-Query로 1분마다 자동 refetch

프로젝트 관리 모듈

- 참여자 정보 전체 CRUD 구현:
    - React-Query mutation 활용 (POST, PATCH)
    - 6가지 중복체크 및 형식 검증 (랜덤번호, 이름, 전화번호, 생년월일,
    모바일앱ID, 워치S/N)
    - 토글 버튼, Autocomplete 컴포넌트 적용
    - 엑셀 업로드 로딩 처리, 페이지 이탈 방지
- 참여자 상태 관리 시스템:
    - 5가지 상태 (등록, 진행, 중단, 유예, 종료)
    - 툴팁(Tooltip) 적용으로 복잡한 상태 설명 제공
    - LocalStorage 기반 히스토리 관리
- 기관/그룹 관리:
    - 기관 생성, 수정, 삭제
    - 관리자 CRUD (TypeScript, 정규표현식 폼검증)

채팅 시스템 통합 (10월)

- Stream 채팅 솔루션 연동
- 실시간 메시지 기능 (메시지 전송, 파일 전송, 읽기 확인)
- 데스크톱 알림 (Notification API) 구현
- 웹 푸시 클릭 시 해당 채널로 자동 이동
- LocalStorage 활용 중복 알림 방지
- 피그마 디자인 시안 기반 풀스크린 레이아웃

배포 및 인프라 (7월)

- Next.js 빌드 환경 구축 (ESLint, Babel, next.config.js)
- Docker 설정 가이드 작성 및 공유
- 메뉴별 아이콘 부여, 페이지 위치 표시 개선
