Android 앱 개발 (Kotlin, Jetpack Compose)

- WebView 기반 하이브리드 앱 아키텍처 설계 및 구현
- FCM 푸시 알림 시스템 구현 (잠금화면/백그라운드/포그라운드 모든 상태 지원)
- Node.js 서버와 FCM SDK 연동으로 실시간 낙상 감지 알림 구현
- WebView-Native 양방향 통신 인터페이스 (JavaScript Interface) 구현
- 토큰 기반 인증 및 로그인 세션 유지 (SharedPreferences)
- Lottie 애니메이션 활용한 스플래시 화면 구현 (3초 프리로딩)
- WebView 캐싱 전략으로 성능 최적화
- 뒤로가기 네비게이션 최적화 (웹 히스토리 우선, 종료 방지)
- 스와이프 새로고침 기능 구현
- 불안정한 네트워크 환경 대응 로직 (연결 복구 시 자동 리로드)
- 앱 종료 시 쿠키 삭제 버그 해결
- Google Play 심사 통과 및 배포 완료

iOS 앱 개발 (SwiftUI)

- Xcode 개발 환경 구축
- iOS 네이티브 앱 개발 및 App Store 배포
- Safari WebView 버그 수정:
    - 아이디 입력 시 2초 후 초기화 문제 해결
    - 첫 글자 자동 대문자 변환 문제 해결
    - 첫 글자 입력 안 되는 문제 해결
    - 로그인 화면 전체화면 미표시 문제 해결
- Android-iOS 크로스 플랫폼 일관성 유지 및 사이드이펙트 디버깅

앱 스토어 심사 및 정책 준수
